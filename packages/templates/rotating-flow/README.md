# rotating-flow runtime template

This Remotion project is packaged with the `yumoframe` CLI. It receives the active project's generated `project.json` through Remotion input props and uses `YUMOFRAME_PROJECT` as its public media directory.

Normal users run `yumoframe dev` and `yumoframe render`; they do not install or invoke this nested package directly. After `yumoframe eject`, edit this source in place while continuing to use the CLI for dependency resolution and rendering.
